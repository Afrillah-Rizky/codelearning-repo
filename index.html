Introduction

HTML and CSS is a languages that been used for creating a website in front-end side (Client side). HTML stands for Hyper-text Markup Language, is it a used to give structure of your website. meanwhile CSS ore Cascading Style sheets is language that can be use to style the front-end of the website. Learning HTML and CSS is a fundamental skill for front-end developer to design the website, and make it to become interesting and intuitive for users to acces the website.

Terms

Here is the terms that commonly used in HTML and CSS language

HTML terms

  • Element : A fundamental building block in HTML, defined by tags, such as p for paragraphs or h1 for header
  • Tag : an HTML code that defines an element. it consist of an opening tag, content, and a closing tag.
  • Attribute : extra information added to HTML elements to provide additional details or modify their behavior
  • Attribute Value : the value assigned to an attribute.
  • HTML Document : the complete structure of an HTML page, include DOCTYPE Declaration, head and body
  • DOCTYPE Declaration : specifies the HTML version being used and must be declared at the begining of an HTML document
  • HTML Head : containt meta-information about HTML document
  • HTML Body : containt the content of the HTML document
  • Tag Nesting : placing one HTML element inside another
  • HTML Comment : a way to add Commentto the HTML code that won't displayed on the webpage.

CSS Terms

  • Selector = pattern that is used to select and style HTML elements on a webpage
  • Property = an attribute of a CSS rule that defines the visual or behavior of the selected elements
  • Value = specific settings assigned to CSS property
  • Declaration = combination of a property and its associated value within a CSS Rule
    example:

    selector {
      property: value;
    }
  • Class = a way to group and apply styles to multiple HTML elements, it is one of selector.
  • ID = unique identifier for a specific HTML element on a webpage

Basic HTML Elements

Creating website is like creating a building, so HTML is a language to create a structure of your website

all html elements below written using tags "<>"

  • !DOCTYPE html = Declaration to specify the HTML Version
  • html = root elements that wraps all contents on the page
  • head = Containts metadata about HTML documents, such as title, charset, and linked stylesheet
  • title = Sets the title of the HTML document (will be displayed in the browset tab)
  • meta charset="UTF-8" = Declares that the character encoding of the HTML documents(Commonly use UTF-8)
  • body = contains the content of the HTML document, include text, image links, etc
  • h1 to h6 = heading elements for define different levels of heading from highest level to lowest
  • p = paragraph element for text content
  • a = anchor element for creating hyperlink
  • etc = you can learn any other element by visiting reference
Basic CSS Elements

using same analogy in HTML, CSS is a language to style your website

Below is basic elements of css that oftenly used

  1. Selector

    2. css selectors target HTML elements for styling

    • 'element' selector(e.g., 'p' for paragraph)
    • .classselector(e.g., .container for elements with the class="container")
    • #id selector (e.g., #header for an element with the id = )
  2. Color
    • 'Color' : Set text color
    • 'background-color': set the background color
  3. Typography
    • 'font-family' : Sets the font family
    • 'font-size' : Sets the font size
    • 'font-weight' : Sets the font weight(e.g., 'bold')
  4. Text
    • 'text-align' : Align text (e.g., 'left', 'center', 'right')
  5. Margins and Paddings
    • 'margin' : Sets the margin outside the element
    • 'Paddings' : Sets the Padding inside the element
  6. Border
    • 'border' : Sets the border properties(e.g., '1px solid #000')
  7. Width and Height
    • 'width' : Sets the width of an element
    • 'height' : Sets the height of an element
  8. Display
    • 'display' : Specifies how an element is displayed(e.g., 'block', 'inline', 'none'
  9. Positioning
    • 'position' : Sets the position method (e.g., 'relative', 'absolute', 'fixed')
    • 'top', 'right', 'bottom', 'left' : positions element
  10. Flexbox
    • 'display: flex' : Enables flexbox layout for the countainer
    • 'flex-direction' : Sets the direction of the flex container
    • 'justify-content' : Align items along the main axis
    • 'align-items' : Align items along the cross axis
  11. Grid
    • 'display: grid' : Enables grid layout for the container
    • 'grid-template-columns','grid-template-rows' : Defines the grid structures
  12. Background
    • 'background-image' : Sets the background image
    • 'background-size', 'background-postion' : Controls background image properties
