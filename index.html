Introduction

HTML and CSS is a languages that been used for creating a website in front-end side (Client side). HTML stands for Hyper-text Markup Language, is it a used to give structure of your website. meanwhile CSS ore Cascading Style sheets is language that can be use to style the front-end of the website. Learning HTML and CSS is a fundamental skill for front-end developer to design the website, and make it to become interesting and intuitive for users to acces the website.

Setup Environment

What you need to start coding with HTML & CSS

  • Text Editor

    Choose a text editor for wrinting your HTML and CSS Code

    Example of text editor :

    • Visual Studio Code : lightweight, free and open source text editor with a lot of feature to help for Web development
    • Atom : a customizable and user-friendly text editor

  • Web Browser

    Have a modern web browser installed for testing and previewing your HTML and CSS. Common choices include Google Chrome, Mozilla Firefox, or Microsoft Edge.

  • HTML and CSS File

    Create your HTML file (e.g., index.html) and CSS file (e.g., style.css) in your project folder.

  • Additional : Live Server extention on VS Code

    If you use VS Code as an text editor, Live Server extention is a great tool to display your code in your selected browser

Basic HTML Elements

Creating website is like creating a building, so HTML is a language to create a structure of your website

all html elements below written using tags "<>"

  • !DOCTYPE html = Declaration to specify the HTML Version
  • html = root elements that wraps all contents on the page
  • head = Containts metadata about HTML documents, such as title, charset, and linked stylesheet
  • title = Sets the title of the HTML document (will be displayed in the browset tab)
  • meta charset="UTF-8" = Declares that the character encoding of the HTML documents(Commonly use UTF-8)
  • body = contains the content of the HTML document, include text, image links, etc
  • h1 to h6 = heading elements for define different levels of heading from highest level to lowest
  • p = paragraph element for text content
  • a = anchor element for creating hyperlink
  • etc = you can learn any other element by visiting reference
